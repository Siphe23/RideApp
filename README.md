Ride Preference App

A modern ride-hailing app concept focused on privacy, safety, and user preference.
This project introduces a smart driver preference system where users can:

Save favorite drivers
Avoid specific drivers privately
Switch between Fast Mode and Relaxed Mode
Improve rider comfort without affecting driver ratings
📱 Project Overview

Traditional ride-hailing apps focus mainly on speed and automatic matching.
This project explores a more personalized experience by adding a Smart Preference Layer on top of normal ride matching.

Users can:

Select trusted drivers
Quietly avoid certain drivers
Prioritize comfort and privacy
Still keep the normal fast ride system intact
✨ Features
⭐ Favorite Drivers

Users can save up to 3 favorite drivers for future rides.

🚫 Private Avoid List

Users can quietly avoid specific drivers without affecting their ratings or accounts.

⚡ Fast Mode

Normal ride-hailing experience with instant matching.

🧘 Relaxed Mode

Attempts to match preferred drivers first before using normal matching.
